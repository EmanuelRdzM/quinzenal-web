# Quinzenal Web 📊

Frontend del proyecto **Quinzenal (CashFlow)**.
Interfaz web para la gestión y visualización del flujo de dinero personal.

## 🚀 Stack
- Vue 
- Vite / Next.js
- TailwindCSS
- Axios

## 📦 Funcionalidades
- Dashboard de balance quincenal
- Registro rápido de ingresos y gastos
- Separación efectivo / tarjeta
- Historial de tarjetas
- Control de deudas y créditos
- Visualización clara de saldos

npm install
